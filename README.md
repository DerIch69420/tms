# tms - Tmux Script

tms is an easy to use interpreted language to make the creation of new tmux sessions simpler

---

## Use

### Clone the repo
```
git clone https://github.com/DerIch69420/tms.git
```

### Build tms 
```
cd tms 
cargo build --release
```

### Copy to env
```
cp target/release/tms ~/.local/bin/
```
